Haskell Cheat Sheet

This document serves as a concise reference for essential functions, operations, and concepts in Haskell programming language.

  1. succ: Increments of 1.
  2. min: Returns minimum.
  3. max: Returns maximum.
  4. head (for lists): First element.
  5. tail (for lists): All elements but first.
  6. last (for lists): Last element.
  7. init (for lists): All elements but last.
  8. length (for lists): List's elements count.
  9. null (for lists): Check if list is empty.
  10. reverse (for lists): Reverse the list.
  11. take (for lists): Extracts that many elements from the beginning of the list.
  12. drop (for lists): Drops the number of elements from the beginning of the list.
  13. minimum (for lists): Returns the smallest element.
  14. maximum (for lists): Returns the biggest element.
  15. sum (for lists): Returns sum of elements.
  16. product (for lists): Multiply all elements.
  17. elem (for lists): Check if the given thing is an element of the list.
  18. cycle: Takes a list and cycles it into an infinite list.
  19. repeat: Takes an element and produces an infinite list of just that element.
  20. replicate: Replicates some number of the same element in a list.
  21. fst (for tuples): Returns tuples first component.
  22. snd (for tuples): Returns tuples second component.
  23. zip: It takes two lists and then zips them together into one list by joining the matching elements into tuples.
